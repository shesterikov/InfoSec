# Практическая работа 2
## Восстановления паролей из хешей

### Цель работы
Познакомиться с процессом восстановления паролей из хеша с использованием инструмента Hydra и словаря

### Технические требования:
- Наличие интернет-соединения
- Наличие [Docker](https://docs.docker.com/desktop/)

### Ход работы:
1. Выполните загрузку Docker образа Kali Linux на локальную машину `docker pull kalilinux/kali-rolling`

2. Выполните запуск Docker контейнера с образом Kali Libux `docker run -v ./data:/app/data --tty --interactive kalilinux/kali-rolling`
   > Данная команда содержит в себе флаг `-v ./data:/app/data` для монтирования директории `data` практической работы из хост системы внутрь Docker контейнера (директория `/app/data`) со всем содержимым данной директории.

3. Выполните обновление списка доступных зависимостей `apt-get update`

4. Выполните установку зависимостей `apt install hashcat wordlists`
    > При наличии ошибок запустите команду выше с флагом `--fix-missing`

5. Перейдите в каталог `cd /usr/share/wordlists/` и распакуйте архив при помощи команды `gunzip rockyou.txt.gz`

6. Перейдите в директорию, содержащую файл с хешами `cd /app/data`

7. Выполните следующую команду для восстановления пароля из хеша `hashcat -m 0 -a 0 -o cracked.txt --potfile-disable hashes.txt /usr/share/wordlists/rockyou.txt`

8. Отобразите результат выполнения команды `cat cracked.txt`
