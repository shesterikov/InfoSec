# Компьютерная безопасность и защита информации  
## Направления подготовки  
Курс предназначен для студентов следующих направлений подготовки:  
- Математическое обеспечение и администрирование информационных систем;  
- Информатика и вычислительная техника;  
- Программная инженерия.  

> При внесении корректировок с учётом уже усвоенного материала курс может быть адаптирован для студентов других IT-направлений.  

## Связь с другими дисциплинами  
Курс тесно связан со следующими дисциплинами:  
- Веб-программирование и мобильные приложения;  
- Основы профессионального программирования;  
- Анализ требований, тестирование и обеспечение качества программного обеспечения;  
- Разработка проектов в области компьютерных наук;  
- Операционные системы;  
- Основы управления данными;  
- Организация электронных вычислительных машин;  
- Введение в разработку веб-сайтов;  
- Современные технологии программирования;  
- Проектирование и разработка приложений для мобильных устройств;  
- Компьютерные сети и сетевая безопасность.
